📡 Postback Setup

✅ Postback কী এবং কেন দরকার?
Daraz এ কেউ আপনার Affiliate Link এ click করে কিছু কিনলে, Daraz আপনার Server এ একটি notification পাঠায় — এটাই Postback। এই notification এ থাকে: কত টাকার order, কত commission, কোন product, কোন platform থেকে।

Postback ছাড়া আপনাকে manually Daraz Adsense এ গিয়ে দেখতে হবে।
Postback দিয়ে সাথে সাথে Telegram notification আসবে! 🎉
🔗 Order Postback URL
এই URL টি Daraz Adsense → Postback URL Tab এ দিন (Type: Order)
https://YOUR-APP.vercel.app/api/postback/order?offer_id={_p_offer}&pay_amount={_p_pay_amount}&payout={_p_payout}&item_id={_p_a2c_item_id}&category_l1={_p_cate_name_lv1}&device={_device_os}&channel={sub_aff_id}&txn_id={sub_id1}&platform={sub_id2}&campaign={sub_id3}&user_id={sub_id4}&date={sub_id5}
🔗 PDP View Postback URL
Daraz Adsense → Postback URL Tab (Type: PDP View)
https://YOUR-APP.vercel.app/api/postback/pdpview?offer_id={_p_offer}&item_id={_p_viewed_item_id}&view_time={_p_pdp_view_time}&category_l1={_p_cate_name_lv1}&device={_device_os}&channel={sub_aff_id}&platform={sub_id2}&user_id={sub_id4}
📋 Tracking Link Structure
আপনার tracking link এ এগুলো automatically যোগ হবে:
Parameterকী পাঠাবে
sub_aff_idআপনার Affiliate ID
sub_id1Unique Transaction ID
sub_id2Platform নাম
sub_id3Campaign নাম
sub_id4User ID
sub_id5তারিখ (YYYY-MM-DD)
⚠️ সাধারণ Error ও সমাধান
❌ invalid click id
Test value ব্যবহার করবেন না। Real URL দিন।
⚠️ broken clickid
txn_id এর জন্য sub_id1 ব্যবহার করুন, sub_aff_id নয়।
ℹ️ SSL Error
Vercel ব্যবহার করলে automatically ঠিক থাকবে।
✅ D+1
Sale হলে পরের দিন Postback আসবে। ধৈর্য রাখুন।

📋 Step-by-Step Setup Guide

একদম শুরু থেকে শেষ পর্যন্ত — বিগিনারদের জন্য
🔴 Phase 1 — Account তৈরি (আজকেই করুন, ৩০ মিনিট, সম্পূর্ণ বিনামূল্যে)
1

Telegram Bot তৈরি করুন

Telegram অ্যাপে @BotFather সার্চ করুন → /newbot লিখুন → Bot এর নাম দিন: DarazDealBD → Username দিন: DarazDealBD_bot → একটি Token পাবেন (যেমন: 1234567:ABCDEFxyz) → এটি কোথাও save করুন। এই Token ই আপনার Bot এর চাবি।

2

আপনার Telegram Chat ID জানুন

Telegram এ @userinfobot সার্চ করুন → /start দিন → আপনার Chat ID দেখাবে (যেমন: 987654321) → এটিও save করুন। Sale হলে এই ID তে notification আসবে।

3

GitHub Account তৈরি করুন

github.com এ যান → Sign Up করুন (বিনামূল্যে) → Email verify করুন। এখানে আপনার code রাখা হবে।

4

Supabase Account তৈরি করুন (Database)

supabase.com এ যান → "Start your project" → GitHub দিয়ে login করুন → "New project" → Name: daraz-affiliate → Password দিন → Region: Southeast Asia (Singapore) → "Create project" চাপুন। ২ মিনিট অপেক্ষা করুন।

5

Vercel Account তৈরি করুন (Hosting)

vercel.com এ যান → "Start Deploying" → GitHub দিয়ে login করুন। এখানে আপনার app host হবে — বিনামূল্যে।

🔵 Phase 2 — Database Setup (১৫ মিনিট)
6

Supabase এ Database তৈরি করুন

Supabase Dashboard → বাম পাশে SQL Editor → "New Query" → নিচের "💻 Bot Code" page থেকে schema.sql এর সব code copy করুন → এখানে paste করুন → "Run" বোতাম চাপুন। "Success" দেখালে হয়ে গেছে।

7

API Keys নিন

Supabase → Settings (বাম নিচে) → API → দুটি জিনিস copy করুন:
১. Project URL: https://xxxxx.supabase.co
২. service_role key: eyJhbGci... (Secret বলে লেখা থাকবে)

🟢 Phase 3 — App Deploy করুন (২০ মিনিট)
8

Code GitHub এ আপলোড করুন

GitHub → "New repository" → Name: daraz-affiliate-bot → Public → Create.
তারপর downloaded zip থেকে সব files এই repository তে upload করুন (drag & drop কাজ করে)।

9

Vercel এ Deploy করুন

Vercel → "Add New Project" → GitHub repository বেছে নিন → Environment Variables section এ নিচের সব variable যোগ করুন:

TG_BOT_TOKEN = আপনার Telegram Token
TG_ADMIN_CHAT_ID = আপনার Chat ID
SUPABASE_URL = Supabase Project URL
SUPABASE_SERVICE_KEY = Supabase Service Key
DARAZ_AFF_ID = আপনার Daraz Affiliate ID

→ "Deploy" চাপুন। ২ মিনিটে live হবে। URL পাবেন: https://your-app.vercel.app

10

Telegram Webhook Set করুন

Browser এ এই URL খুলুন (আপনার info দিয়ে replace করুন):

https://api.telegram.org/botYOUR_TOKEN/setWebhook?url=https://your-app.vercel.app/api/telegram

{"ok":true} দেখালে Bot চালু হয়ে গেছে! 🎉

11

Daraz Adsense এ Postback URL দিন

adsense.daraz.com.bd → Login → Postback URL Tab → "Add New" →
Name: Order Tracker → Type: Order → URL: (📡 Postback page থেকে copy করুন) → Save.

একইভাবে PDP View এর জন্যও যোগ করুন।

🟣 Phase 4 — পণ্য যোগ ও Marketing শুরু
12

প্রথম পণ্য যোগ করুন

এই Admin Panel এর "➕ যোগ করুন" Tab এ যান → Daraz থেকে পছন্দের পণ্যের link কপি করুন → Form পূরণ করুন → Save করুন। পণ্যের original price ও current price দিলে automatically discount badge দেখাবে।

13

Telegram Bot Test করুন

আপনার Bot খুলুন → /start দিন → /search মোবাইল দিন → পণ্য দেখা গেলে সব কাজ করছে। এবার Bot link বন্ধু-বান্ধবদের শেয়ার করুন।

14

Facebook/WhatsApp এ শেয়ার শুরু করুন

"🔗 Links" Tab এ যান → পণ্যের link দিন → Platform বেছে নিন → Generated link কপি করুন → Facebook/WhatsApp এ post করুন। প্রতিটি platform এর জন্য আলাদা link তৈরি করুন।

15

প্রথম Sale আসলে 🎉

Sale হলে Daraz পরের দিন (D+1) আপনার Postback URL এ notification পাঠাবে। আপনি সাথে সাথে Telegram এ notification পাবেন। Daraz Adsense → Reports এ সব দেখতে পাবেন।

// ══════════════════════════════════════════════════ // DARAZ AFFILIATE TELEGRAM BOT — FINAL VERSION // ══════════════════════════════════════════════════ const TelegramBot = require('node-telegram-bot-api') const { createClient } = require('@supabase/supabase-js') const bot = new TelegramBot(process.env.TG_BOT_TOKEN, { polling: true }) const db = createClient(process.env.SUPABASE_URL, process.env.SUPABASE_SERVICE_KEY) // ── Affiliate Link ────────────────────────────────── function affLink(url, platform, userId) { try { const u = new URL(url) u.searchParams.set('sub_aff_id', process.env.DARAZ_AFF_ID) u.searchParams.set('sub_id1', `txn_${Date.now()}`) u.searchParams.set('sub_id2', platform) u.searchParams.set('sub_id3', 'organic') u.searchParams.set('sub_id4', String(userId)) u.searchParams.set('sub_id5', new Date().toISOString().split('T')[0]) return u.toString() } catch { return url } } // ── Discount ──────────────────────────────────────── function disc(p) { const o = +p.original_price, c = +p.price if (o > c) { const saved = o - c, pct = Math.round((saved/o)*100) return { hasDeal:true, saved, pct, o, c } } return { hasDeal:false, saved:0, pct:0, o:c, c } } // ── Format Card ───────────────────────────────────── function card(p, userId) { const d = disc(p) const link = affLink(p.daraz_link, 'telegram', userId) const stars = '⭐'.repeat(Math.round(p.rating||4)) const badge = d.hasDeal ? `🔥 ৳${d.saved.toLocaleString()} ছাড় (${d.pct}% OFF)` : '✅ সেরা দামে পাচ্ছেন' const txt = `🛍️ *${p.name}*\n` + `${'─'.repeat(28)}\n` + (d.hasDeal ? `💵 আগের দাম: ~৳${d.o.toLocaleString()}~\n✅ এখন: *৳${d.c.toLocaleString()}*\n` : `💵 মূল্য: *৳${d.c.toLocaleString()}*\n`) + `${badge}\n` + `${stars} ${p.rating||4.5}\n` + (p.brand ? `🏷️ ${p.brand}\n` : '') + `${'─'.repeat(28)}\n` + `📂 ${p.category_label||p.category}` return { txt, link } } // ── Send Product ──────────────────────────────────── async function send(chatId, p, userId) { const { txt, link } = card(p, userId) const kb = { inline_keyboard: [[ { text: '🛒 Daraz এ কিনুন', url: link } ],[ { text: '📤 শেয়ার করুন', switch_inline_query: p.name } ]]} try { if (p.image_url) { await bot.sendPhoto(chatId, p.image_url, { caption:txt, parse_mode:'Markdown', reply_markup:kb }) } else { await bot.sendMessage(chatId, txt, { parse_mode:'Markdown', reply_markup:kb }) } } catch { await bot.sendMessage(chatId, txt, { parse_mode:'Markdown', reply_markup:kb }) } } // ── DB Helpers ────────────────────────────────────── const search = async q => (await db.from('products').select('*').eq('in_stock',true) .ilike('name',`%${q}%`).order('discount_amount',{ascending:false}).limit(5)).data || [] const byCat = async cat => (await db.from('products').select('*').eq('in_stock',true) .eq('category',cat).order('discount_amount',{ascending:false}).limit(6)).data || [] const deals = async () => (await db.from('products').select('*').eq('in_stock',true) .gt('discount_amount',0).order('discount_amount',{ascending:false}).limit(5)).data || [] // ── /start ────────────────────────────────────────── bot.onText(/\/start/, msg => { bot.sendMessage(msg.chat.id, `👋 *স্বাগতম ${msg.from.first_name||'বন্ধু'}!*\n\n` + `🛍️ Daraz এর সেরা ডিসকাউন্ট এখানে পাবেন!\n\n` + `🔍 /search [পণ্যের নাম] — খুঁজুন\n` + `🔥 /deals — আজকের সেরা ছাড়\n` + `📂 /category — ক্যাটাগরি\n` + `❓ /help — সাহায্য\n\n` + `💡 _উদাহরণ: /search ব্লুটুথ হেডফোন_`, { parse_mode:'Markdown' }) }) // ── /deals ────────────────────────────────────────── bot.onText(/\/deals/, async msg => { const ps = await deals() if(!ps.length) return bot.sendMessage(msg.chat.id,'😔 এখন কোনো বিশেষ অফার নেই।') bot.sendMessage(msg.chat.id, `🔥 *${ps.length}টি সেরা অফার!*`, {parse_mode:'Markdown'}) for(const p of ps) { await send(msg.chat.id, p, msg.from.id) await new Promise(r => setTimeout(r, 400)) } }) // ── /search ───────────────────────────────────────── bot.onText(/\/search (.+)/, async (msg, m) => { const ps = await search(m[1].trim()) if(!ps.length) return bot.sendMessage(msg.chat.id, `😔 *"${m[1]}"* পাওয়া যায়নি।\n\n💡 /search মোবাইল`, {parse_mode:'Markdown'}) bot.sendMessage(msg.chat.id, `✅ *${ps.length}টি পণ্য পাওয়া গেছে!*`, {parse_mode:'Markdown'}) for(const p of ps) { await send(msg.chat.id, p, msg.from.id) await new Promise(r => setTimeout(r, 400)) } }) bot.onText(/^\/search$/, msg => bot.sendMessage(msg.chat.id, '🔍 এভাবে লিখুন:\n`/search মোবাইল`', {parse_mode:'Markdown'})) // ── /category [{text:'📱 মোবাইল',callback_data:'c:mobile'},{text:'💻 ইলেকট্রনিক্স',callback_data:'c:electronics'}], [{text:'👗 ফ্যাশন',callback_data:'c:fashion'},{text:'👟 জুতা ও ব্যাগ',callback_data:'c:shoes'}], [{text:'🏠 হোম ও লিভিং',callback_data:'c:home'},{text:'🍳 কিচেন',callback_data:'c:kitchen'}], [{text:'💄 বিউটি',callback_data:'c:beauty'},{text:'🧸 শিশু ও খেলনা',callback_data:'c:kids'}], [{text:'🔥 সব ডিসকাউন্ট',callback_data:'c:deals'}] ]} }) }) bot.on('callback_query', async q => { if(!q.data.startsWith('c:')) return await bot.answerCallbackQuery(q.id) const cat = q.data.replace('c:','') const ps = cat === 'deals' ? await deals() : await byCat(cat) if(!ps.length) return bot.sendMessage(q.message.chat.id,'😔 এই ক্যাটাগরিতে পণ্য নেই।') bot.sendMessage(q.message.chat.id, `✅ *${ps.length}টি পণ্য*`, {parse_mode:'Markdown'}) for(const p of ps) { await send(q.message.chat.id, p, q.from.id) await new Promise(r => setTimeout(r, 400)) } }) // ── /help ─────────────────────────────────────────── bot.onText(/\/help/, msg => bot.sendMessage(msg.chat.id, `❓ *সাহায্য*\n\n` + `\`/search মোবাইল\` — পণ্য খুঁজুন\n` + `\`/deals\` — সেরা ছাড়\n` + `\`/category\` — ক্যাটাগরি মেনু\n\n` + `🛒 পণ্যের নিচে বাটনে ক্লিক করুন`, { parse_mode:'Markdown' })) // ── Plain text → search hint ──────────────────────── bot.on('message', msg => { if(msg.text && !msg.text.startsWith('/') && msg.text.length > 2) bot.sendMessage(msg.chat.id, `🔍 এভাবে খুঁজুন:\n\`/search ${msg.text}\``, { parse_mode:'Markdown' }) }) console.log('🤖 Daraz Affiliate Bot চালু!')

-- ════════════════════════════════════════════ -- DARAZ AFFILIATE DATABASE — FINAL VERSION -- Supabase SQL Editor এ paste করে Run করুন -- ════════════════════════════════════════════ CREATE TABLE IF NOT EXISTS products ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), name TEXT NOT NULL, price DECIMAL(10,2) NOT NULL, original_price DECIMAL(10,2), discount_amount DECIMAL(10,2) GENERATED ALWAYS AS (GREATEST(original_price - price, 0)) STORED, discount_percent INT GENERATED ALWAYS AS (CASE WHEN original_price > 0 THEN ROUND(((original_price-price)/original_price)*100) ELSE 0 END) STORED, image_url TEXT, daraz_link TEXT NOT NULL, category TEXT NOT NULL, category_label TEXT, rating DECIMAL(3,1) DEFAULT 4.5, commission DECIMAL(5,2) DEFAULT 0, brand TEXT, in_stock BOOLEAN DEFAULT TRUE, clicks INT DEFAULT 0, source TEXT DEFAULT 'manual', created_at TIMESTAMPTZ DEFAULT NOW() );
