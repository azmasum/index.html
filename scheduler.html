⚙️ Daily Auto-Post Setup (Vercel Cron)
প্রতিদিন সকাল ১০টায় auto post করতে:
Step 1: vercel.json এ এটা যোগ করুন:
{
  "crons": [{
    "path": "/api/cron/daily-deals",
    "schedule": "0 4 * * *"
    // UTC 4 = BD সকাল ১০টা
  }]
}
Step 2: Vercel → Settings → Environment Variables এ যোগ করুন:
TG_CHANNEL_ID = @darazme
TG_GROUP_ID = @DarazDealBDBD
CRON_SECRET = daraz2024secret
Manual Test করুন:
Deploy হলে এই URL এ GET request করুন:
https://daraz-affiliate-app.vercel.app/api/cron/daily-deals?type=daily
⚠️ Channel/Group এ Bot Admin করুন:
→ @darazme Channel এ @DarazDealBD_bot কে Admin করুন
→ @DarazDealBDBD Group এ @DarazDealBD_bot কে Admin করুন
